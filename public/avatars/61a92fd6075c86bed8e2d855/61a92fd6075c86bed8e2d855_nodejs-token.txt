Для работы с токеном совершаются такие этапы (с использованием middleware 'authenticate')
1. Извлекается токен из заголовка (req.headers.authorization)
 - строка "Bearer jenrgkdfngk.egwtgwrtg.ertyerty" превращается в массив по пробелу
2. Токен проверяется на валидность
 - Проверяется, является ли первое слово = "Bearer"
 - проверяем,  мы ли выдали токен (jwt.verify(token, SECRET))
3. Находим в базе пользователя по id из токена
4. Прикрепляем токен к телу запроса (req)
5. Обраьотка запроса передается далее
